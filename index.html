Easy Installation

Get started in minutes with these simple installation methods

Quick Install (Recommended)

Run this one-line command in your terminal:

bash <(curl -sL https://raw.githubusercontent.com/OWASP-BLT/BLT-Extension/main/install.sh)
Automatic browser detection and profile selection

Manual Installation

  1. 1. Download from Releases page
  2. 2. Unzip the downloaded file
  3. 3. Open chrome://extensions/
  4. 4. Enable "Developer mode"
  5. 5. Click "Load unpacked" and select the folder
