Open Dedicated Features Page

Screenshot Capture and Upload

Capture visible pages, annotate with the built-in editor, and upload to BLT-compatible flows for bug evidence collection.

Job Application Tracker

Track applications from LinkedIn and Wellfound, store entries in extension local storage, and manage them in the dashboard table.

Export and Status Management

Update statuses, filter records, bulk-manage rows, and export job applications as CSV directly from the tracking panel.

Trademark Scanner

Scan page keywords against BLT trademark API results and highlight terms inline to reduce accidental legal/content mistakes.

GitHub PR Assist Buttons

Get floating quick-action controls on pull request pages to navigate update actions, files sections, and long review pages faster.

BLT Contributor Workflow

Support BLT’s points and contribution journey by making evidence capture and task tracking faster from where users already work.

Quick install

Run this command to detect supported browsers and install to your selected profile.

bash <(curl -sL https://raw.githubusercontent.com/OWASP-BLT/BLT-Extension/main/install.sh)
Supported: Chrome, Chromium, Brave, Microsoft Edge.

Manual install

  1. 1.Download package from GitHub Releases.
  2. 2.Unzip and open chrome://extensions/.
  3. 3.Enable Developer mode and click Load unpacked.
  4. 4.Select the BLT extension directory.
