Screenshot Capture + Annotation

Capture visible or selected areas, annotate, and upload/share evidence for bug reporting workflows.

  • Toolbar editing tools: pencil, line, arrow, rectangle, marker, text, color, undo.
  • Upload and sharing actions integrated in screenshot flow.
  • Primary files: content.js, event.js, screenshot.html.

Trademark Scanner

Scans page keywords, queries BLT trademark API, and highlights risky terms inline.

  • Runs from popup action and page message listener.
  • Highlights terms with visible marked style in content.
  • Primary files: trademark-scanner.js, background.js.

Job Monitoring + Dashboard

Tracks applications on LinkedIn and Wellfound, stores entries, and manages them in a dashboard.

  • Auto-capture application metadata with MutationObserver-based hooks.
  • Status updates, pagination, and CSV export included.
  • Primary files: linkedin-monitor.js, wellfound-monitor.js, jobtracking.html, jobtracking.js.

GitHub PR Assistant

Adds floating navigation buttons on PR pages for faster review and branch-update navigation.

  • PR page helper UI with discoverable quick actions.
  • Targets update branch and review navigation points.
  • Primary file: github-pr-button.js.
